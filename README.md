# 数择云平台

## 关于我们

数择云平台是**开源一站式大数据智能云研发平台**。 同时能满足用户对数据治理的需求，赋予用户对外提供数据服务的能力。



数择云平台是从工作室、车间到工具集都齐备的一站式大数据工场，助力您快速完成数据集成、开发、治理、服务、质量和安全等全套数据研发工作。产品功能对标阿里云DataWorks，我们的目标是做开源界最好用的**DataWorks**。
